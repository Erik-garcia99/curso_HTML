sintaxis de imagenes en HTML

el HTML <img> se utiliza para incrustar una imagen dentro de una pagina web esta etiqeuta esta vacia, solo contiene atributos que hacen referencia a la imageny solo tiene etiqeuta de apertura mas no de cierre

esta requiere dos atributos

  • src: especifica la ruta de la imagen
    esta puede ser un enlace absoluto o relativos

  • alt: especifica un texto alternativo a la imagen
    le da mayor importancia al SEO del navegador y util para herramientas de lectura de pantalla

sintaxis: <img src="url" alt="textto_alterno">

atributo src

el atributo src especifica la ruta de la imagen
esta imagen lo ideal es que se encuentro de la carpteta del proyecto, que se acceda medinate una ruta relativa, ya que es mucho mas facil de trarla a excepticon que se decida hacer uso de una imagen en intrenet lo cual lo ideal es agregar el alt en caso que la imagen no se encuentr disponible

atributo alt

alt: este atrobuto proporciona un texto alternativo para una imagen si es usuario por alguna razon no puede verlo debe de ser un texto corto descriptivo

tamanio de la imagen - ANCHO y ALTURA

para darle el tamanio de este se hace mediante el CSS el cual tiene 2 propiedades pra esto el Width que establece el ancho de la imagen y el height que establece el alto, lo ideal es solo usar 1 usar solo el ancho o solo el alto, porque si se usan los 2 cunado se haga mas peqnia la pantalla o mas grande la imagen se puede deformar y si solo se usa 1 el otro lado de establece por automatico lo ahce el navegador. ejemplo: seria dentro del codigo <style>{ img{ width:500px; height:600px; } } css tiene varios unidades de medicion entre los mas comines son los pixeles (px) y los em

la etiqeuta permite GIFS, usar una imagen como un enlace, esto se hace poniendo la imagen dentro de un tag <a>
