que es HTML

HTML es el estnadar para el lenguaje estantar para la creacion de paginas web. HTML decribe la estrucutra de la pagina web, define los elementos, que es parrafo, es que un enlace. etc.

si entras a esta pagina a ver el codigo se puede ver como esta compuesto, los elemntos basicos son un DOCTYPE esto indica el tipo de documento, esto se queda de la epoca en donde los etandares no eran lo comun en la web. teneemos al (head): este es el encabezado de la pagina, aqui se colocan los metadatos, los estilos, los scripts, etc. (body): este es el cuerpo de la pagina, aqui se coloca todo lo que se va a mostrar en la pagina web.

que es un elemtno en HTML

un elementos en HTML esta dfinido por un tag de inicio siempre, pero no todos tiene uno final, a estos se les conoce como elementos vacios. estos tiene la sigueinte sintaxis:

<tag>contenido</tag>
todo elementos que forme la estrucutura de HTML debe de tener un tag de inicio y uno de cierre, si no se tiene un tag de cierre se debe de cerrar con un slash al final del tag de inicio.
