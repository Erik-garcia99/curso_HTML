favicon

el favicon es una imagen pequenia que esta antes del titulo de nuestra pagina en la pestalla del navegador

estos se pueden crear en la siguiente web

https://www.favicon.cc

o se puede descargar o mandar a disenia, estos por lo general tienen el formado .ico

para agregar este favicon, es necesario que este guardado en alguna lugar de nustra capeta de proyecto, la froma en como se une es mecinate la etiqueta <link>, dentro de la etiqueta <header> despues del titulo

algunos navegadores aceptan los sigueintes formatos

HTML page title

todas las paginas web deben de contener un titulo que las describa

esto se logra con la etiqeuta <title>, este se muestra en la pestania del navegador

este titulo debe de ser algo qeu describa el contenido dentro de la pagina, esto es muy importante para el motor de optimizacion de busqueda (SEO),
