text firmatting

existen varias etiquetas para dar formato al texto dentro de HTML

  • <b> -> bold text: marca de color negro pero este no tiene peso semantico en el documento

  • <strong> -> hace lo mismo de (b) con la diferencia que esta etiqeuta si tiene peso semantica en el codigo es para remarcar algo importante.

  • <i> -> texto en italica, hace como las negras en cursiva

  • <em> ->enfatiza el texto, tiene peso semantico, hace lo imsomo que la italica

  • <mark> -> texto marcado

  • <small> -> hace el texto un poco mas pequenio

  • <del> -> tacha el texto que cubre

  • <ins> -> el texto se subralla por una linea a bajo

  • <sub> hace referencia al subindice como pude ser los atomos de una fomrula quimica

    This is subscripted text.


  • <sup>hace practicamente lo mismo de (sub) pero este en vez de estar abajo esta arriba
